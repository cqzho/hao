# 网盘工具扩展 type:300

1. 请尽量整合在 `panTools` 内
2. 请为扩展编写单独的 `json` 文件
3. 请将 json 添加到 uzAio.json 的 panTools
4. json 样例

```
{
  "name": "扩展名称",
  "codeID": "如果加密请填写，由 uz影视 生成",
  "url": "扩展链接",
  "env":"环境变量名称1##环境变量描述1&&环境变量名称2##环境变量描述2",
  "remark": "",
  "version":1,
  "type":300
}
```
